# Waveform Generator

An Electron app that generates a waveform from a WAV file.

Other audio formats work, however, the app will display an incorrect sample frequency.

## Building

### Development

```bash
$ npm run dev
```

### Release

```bash
# For windows
$ npm run build:win

# For macOS
$ npm run build:mac

# For Linux
$ npm run build:linux
```
